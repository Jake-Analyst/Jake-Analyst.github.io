Personal Financial Management System

AI‑Powered Financial Analytics, Budgeting & Automated Reporting

Data Preparation & Cleaning: The raw M‑Pesa statement was first processed using Excel and Power Query to remove duplicates, handle missing values, standardize date formats, and categorize transactions. This ensured a single, trusted source of truth before loading into the analytics system.

System Architecture: The dashboard was built with Python (Flask) and SQLite for on‑the‑fly data storage and querying. Pandas handles dynamic data profiling, while Chart.js and Tailwind CSS render responsive, real‑time visualizations for income, expenses, savings rate, budget utilisation, and spending patterns.

AI & Financial Intelligence: Integrated with Google Gemini, the system provides automated variance analysis, trend forecasting, anomaly detection, and a conversational financial assistant that answers natural‑language questions using the actual transaction data. A one‑click PDF report (via ReportLab) includes executive summaries, spending breakdowns, fee analyses, and AI‑generated recommendations – all directly from the uploaded file.
