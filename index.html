Personal Finance Dashboard – M-Pesa Data Cleaning (Excel)

Cleaned and structured raw M-Pesa transaction statements in Microsoft Excel using Power Query, transforming messy data into an analysis-ready dataset.

⚠️ Messy raw data

✅ Cleaned data

Problem

Raw financial statements often contain messy and inconsistent data that prevents effective analysis. This project addressed the challenge of converting unstructured M-Pesa transaction records into a usable format for reporting and decision-making.

What I Did

I removed duplicates, standardized formats, extracted useful transaction details from text fields, and categorized transactions into income and expense groups. Power Query was used to automate the cleaning process and make the workflow refreshable for future updates.

Key Insights

The final dataset was structured, consistent, and ready for analysis, dashboard creation, and personal financial reporting.

Tools Used

Microsoft Excel Power Query

HR Analytics SQL Project

Analyzed employee salary, gender, and department data using SQL to uncover pay differences and workforce patterns.

Problem

Organizations often store large amounts of HR data but do not fully use it to guide decision-making. This project addresses that gap by using SQL to explore fairness, compensation trends, and workforce composition.

What I Did

I joined employee salary and demographic tables, calculated average salaries by department and occupation, identified top earners, analyzed gender pay gaps, and grouped employees by age categories to study workforce patterns.

Key Insights

Some departments had significantly higher average salaries, certain roles showed gender pay differences, and younger employees were concentrated in specific occupations while older employees were more common in senior roles.

Employee Attrition Dashboard

Power BI dashboard exploring employee turnover by department, income, job satisfaction, and more.

Problem

Employee attrition increases recruitment costs and reduces productivity. The challenge was to identify the main patterns behind turnover so HR teams could respond proactively.

What I Did

I cleaned and standardized a public HR dataset in Excel, imported it into Power BI, created the necessary measures, and built an interactive dashboard to analyze attrition by department, gender, marital status, salary, overtime, and job satisfaction.

Key Insights

Sales and Research & Development showed the highest attrition, lower income was linked to higher turnover, and salary increase patterns were associated with performance changes.

Personal Financial Management System

Intelligent financial dashboard that transforms M-Pesa statements into actionable insights, with AI-powered analysis, KPI cards, and PDF reports.

Many people struggle to understand their spending habits and manage their finances from raw transaction statements. This project solves that by turning financial data into clear, interactive, and practical insights.
